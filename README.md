📊 Análise de Churn de Clientes

📌 Sobre o Projeto

Este projeto tem como objetivo analisar os fatores que influenciam o cancelamento de clientes (churn) em uma empresa de telecomunicações.

Através de uma análise exploratória de dados (EDA), buscamos identificar padrões de comportamento e gerar insights que possam auxiliar na retenção de clientes e na tomada de decisão estratégica.


🎯 Objetivos

Identificar os principais fatores relacionados ao churn
Entender o perfil dos clientes que cancelam
Gerar insights acionáveis para redução da evasão
Criar uma base sólida para futuros modelos preditivos


🛠️ Tecnologias Utilizadas
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook


📂 Base de Dados

O dataset utilizado contém informações sobre clientes de uma empresa de telecom, incluindo:

Dados demográficos
Informações de contrato
Serviços contratados
Informações de pagamento
Status de churn


🔍 Etapas da Análise
1. Limpeza de Dados
Remoção de valores duplicados
Tratamento de valores ausentes
Ajuste de tipos de dados
2. Análise Exploratória (EDA)
Análise univariada
Análise bivariada com churn
Visualização de padrões e distribuições
Comparações entre grupos de clientes


📊 Principais Insights
📉 Tempo de Permanência (Tenure)
Clientes com menor tempo de contrato apresentam maior taxa de churn
A probabilidade de cancelamento diminui com o tempo


- Os primeiros meses são críticos para retenção.



📄 Tipo de Contrato
Contratos mensais possuem maior churn
Contratos de longo prazo reduzem significativamente o cancelamento


- Fidelização contratual é um forte fator de retenção.



💰Valor Mensal
Clientes com mensalidades mais altas tendem a cancelar mais


- O custo pode não estar alinhado com o valor percebido.



🌐 Tipo de Internet
Clientes com fibra óptica apresentam maior churn


- Possível problema de qualidade, expectativa ou concorrência.



💳 Método de Pagamento
Pagamentos automáticos têm menor churn
Métodos manuais (ex: boleto) têm maior churn


- Reduzir fricção no pagamento melhora retenção.



🛠️ Suporte Técnico
Clientes sem suporte técnico apresentam maior churn


- Suporte é um fator crítico na experiência do cliente.



📦 Serviços Adicionais
Clientes com mais serviços (segurança, backup, etc.) cancelam menos


- Maior engajamento reduz a probabilidade de churn.



🧠 Questões Estratégicas

Com base na análise, as seguintes ações podem ser consideradas:

.Focar na retenção nos primeiros meses
.Incentivar contratos de longo prazo
.Promover pagamento automático
.Investigar problemas no serviço de fibra
.Criar pacotes com serviços adicionais
.Reforçar suporte técnico ao cliente


📈 Possíveis Próximos Passos
Construção de modelo de Machine Learning para previsão de churn
Criação de dashboard interativo (Power BI ou Tableau)
Segmentação de clientes por risco de cancelamento
Testes A/B para estratégias de retenção


📌 Conclusão

A análise mostrou que o churn é influenciado por múltiplos fatores, principalmente:

Tempo de permanência
Tipo de contrato
Serviços contratados
Forma de pagamento

Clientes novos, com contratos mensais e baixo engajamento apresentam maior risco de cancelamento.

Com estratégias direcionadas, é possível reduzir significativamente a evasão e aumentar a retenção de clientes.
